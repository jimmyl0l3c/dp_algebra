# Learning support program for linear algebra

This project is my master thesis. The app is currently only in Czech but is planned to be translated to English in the future.

## Requirements

- **Backend**
  - Python 3.14
  - Dependencies managed by uv, see [pyproject.toml](./backend/pyproject.toml)
- **Frontend**
  - Flutter SDK 3.24.3
  - Dependencies specified in [flutter_app/pubspec.yaml](./flutter_app/pubspec.yaml)
